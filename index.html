FAQ

Common questions

What language do you code in?

JavaScript / Node.js with Discord.js. It's the most popular and well-supported library for Discord bots. If you need Python (discord.py), I can do that too.

Do I need to provide hosting?

Nope. I can help you set up hosting on a VPS or guide you through free options. I'll make sure your bot stays online 24/7.

Can you modify an existing bot?

Absolutely. If you have an existing bot that needs new features or bug fixes, send me the code and I'll take a look.

How do I pay?

PayPal or crypto. 50% upfront, 50% on delivery. For smaller projects under $50, full payment upfront.

Will I get the source code?

Yes, you get full ownership of the code. It's yours to modify, share, or do whatever you want with.
